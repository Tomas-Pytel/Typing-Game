# ⌨️ Typing Game – Semestrálny projekt v C#

Tento projekt je interaktívna aplikácia zameraná na trénovanie rýchlosti a presnosti písania. Bol vytvorený ako semestrálna práca s cieľom demonštrovať praktické využitie jazyka C# a princípov objektovo orientovaného programovania (OOP).

## 🚀 Prehľad aplikácie

Hra testuje používateľa v písaní náhodne generovaných slov v reálnom čase. Hlavným cieľom bolo vytvoriť funkčný herný cyklus, ktorý správne spracováva vstupy od používateľa a dynamicky aktualizuje stav hry (skóre, čas, náročnosť).

## 🏗️ Technické prvky a implementácia

V rámci vývoja som sa sústredil na čistotu kódu a využitie kľúčových vlastností C#:
- **OOP Architektúra:** Rozdelenie logiky do tried (napr. pre správu slov, skóre a časovača) s dôrazom na zapuzdrenie (encapsulation).
- **Event-driven Programming:** Využitie udalostí (Events) a handlerov na spracovanie stlačení kláves a aktualizáciu UI v reálnom čase.
- **Práca s kolekciami:** Efektívna správa dát pomocou `List`, `Dictionary` alebo iných generických kolekcií pre ukladanie slovníka slov.
- **Logika hry:** Implementácia algoritmov pre náhodný výber písmen.
